# 🍏 FitAsistan

Yapay zeka destekli, kişiselleştirilmiş diyet ve sağlıklı yaşam asistanı mobil uygulaması.

## 🚀 Güncel Durum (Şu An Buradayız)
* **Mevcut Aşama:** 3. Hafta Tamamlandı
* **Yapılanlar:** 
  * VKİ hesaplama algoritması ve premium sonuç kartı geliştirildi.
  * Recharts ile kilo takip grafiği ve hedef ilerleme sistemi eklendi.
  * Tüm sayfalardan laboratuvar ödevi içerikleri temizlendi.
  * Veri kalıcılığı için LocalStorage entegrasyonu yapıldı.
* **Sıradaki Görev:** 4. Hafta - Sağlık Günlüğü ve Su Takip Sistemi.
